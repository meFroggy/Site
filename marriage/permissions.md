| Permission:                | Description:                                                                                                         |
|:---------------------------|:---------------------------------------------------------------------------------------------------------------------|
| marriage.marry             | Allows the player to use the command /marry player, when config option Use-Permissions-For-Commands is set as true.  |
| marriage.divorce           | Allows the player to use the command /marry divorce, when config option Use-Permissions-For-Commands is set as true. |
| marriage.tp                | Allows the player to use the command /marry tp, when config option Use-Permissions-For-Commands is set as true.      |
| marriage.msg               | Allows the player to use the command /marry msg, when config option Use-Permissions-For-Commands is set as true.     |
| marriage.admin             | Allows the player to use the /marriage command line.                                                                 |
| marriage.admin.requirering | Allows the player to use the command /marriage requirering [true/false].                                             |
